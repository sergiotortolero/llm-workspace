# Inbox — buzón de contexto crudo

Suelta aquí **cualquier cosa** que quieras que yo absorba: exports de claude.ai, notas de
NotebookLM, transcripciones, bullets sueltos, lo que sea. Sin formato, sin orden.

## Reglas simples
- Un archivo por "volcado". Nómbralo con fecha + tema: `2026-06-18-perfil.md`,
  `2026-06-18-kibo-vision.md`, etc.
- Texto plano o Markdown. Si pegas algo largo, no lo edites — yo lo limpio.
- **No pegues secretos** (contraseñas, tokens, connection strings). Ver `../ONBOARDING.md`.

## Qué pasa después
Cuando me digas "procesa el inbox" (o `/context-sync`), leo todo, te pregunto lo que falte,
y lo destilo a la capa correcta (memoria / CLAUDE.md / context / PRD). Luego puedes archivar
o borrar el crudo.

*(Este README es solo la guía; tus archivos van junto a él.)*
